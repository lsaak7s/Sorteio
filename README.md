# 🎲 DevSorteio

Gere números aleatórios entre um valor mínimo e um valor máximo de forma rápida, simples e intuitiva.

**HTML • CSS • JavaScript**

---

## 📌 Sobre o projeto

O **DevSorteio** é uma aplicação web desenvolvida com **HTML5, CSS3 e JavaScript**, criada para gerar números aleatórios dentro de um intervalo definido pelo usuário.

Basta informar o valor mínimo e o valor máximo, clicar em **Sortear** e o sistema exibirá instantaneamente um número aleatório.

Este projeto foi desenvolvido para praticar manipulação do DOM, eventos, funções, geração de números aleatórios e organização de código em JavaScript.

## 🚀 Como usar

Clone o repositório:

```bash
git clone https://github.com/Isaak7s/DevSorteio.git
```

Entre na pasta do projeto:

```bash
cd DevSorteio
```

Abra o arquivo **index.html** no navegador ou publique o projeto utilizando o GitHub Pages.
